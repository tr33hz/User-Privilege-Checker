# User Privilege Checker

## Описание

Простое веб-приложение для проверки привилегий пользователя в операционной системе Windows с использованием Node.js, node-addon-api (C++) и WinAPI.

## Установка и запуск

1. Клонируйте репозиторий:

   (Командная строка)
   git clone https://github.com/tr33hz/User-Privilege-Checker.git
   cd user-privilege-checker

2. Установите зависимости:

	(Командная строка)
	npm install

3. Соберите модуль

	(Командная строка)
	node-gyp configure
    node-gyp build

4. Запустите батник (server.bat)
